# Raw Data Policy

Place the source handbook PDF in this directory before rebuilding local artifacts.

Expected local file:

```text
data/raw/so-tay-sinh-vien-khoa-48.pdf
```

Raw PDFs may be copyrighted by their original publisher. Before making this
repository public, either confirm redistribution rights for the source PDF or
remove the PDF from Git tracking and keep only this README plus rebuild
instructions.
